Network sniffer 
Analyseur réseau


Protocoles implémentés:
-Ethernet
-IP / ARP
-ICMP / TCP /SCTP /UDP
-Telnet / IMAP / SMTP /POP / FTP / HTTP / BOOTP DHCP / DNS

Les  commutateurs  de  la  ligne  de  commande  implementés : 
-i <interface> 
-o <fichier> 
-f <filtre> 
-v <1..3> 
-u pour l'usage

Affichage : 
-packet en HEX puis l'analyse de chaque couche (inspiré de Wireshark)
